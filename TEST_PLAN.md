# 测试计划

## 测试环境
- 服务器: http://47.81.9.246:80
- 客户端: 微信内置浏览器 / Chrome

## 测试类型

### 1. 单元测试 (已完成)
- Socket事件测试
- 用户注册/登录
- 房间创建/加入

### 2. 集成测试 (待完成)
- 5人完整游戏流程
- 胜负判定逻辑
- 刺客刺杀功能

### 3. 体验测试 (待完成)
- 移动端适配
- 断线重连
- 大量玩家同时操作

## 测试用例

### 核心流程
1. 新用户注册 → 创建房间 → 等待玩家 → 开始游戏
2. 老用户登录 → 加入房间 → 参与游戏
3. 完成任务 → 胜负判定 → 积分更新

### 异常处理
1. 网络断开 → 重连 → 恢复状态
2. 房主离开 → 转移房主
3. 人数不足 → 提示无法开始
